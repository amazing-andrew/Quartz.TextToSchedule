AndrewSmith.Quartz.TextToSchedule
-----------------------------------------------------------------

1. INTRODUCTION
----------------

This is the README file for AndrewSmith.Quartz.TextToSchedule.

This is a library that is used to translate plain english into a schedule that can be used by Quartz.Net.
